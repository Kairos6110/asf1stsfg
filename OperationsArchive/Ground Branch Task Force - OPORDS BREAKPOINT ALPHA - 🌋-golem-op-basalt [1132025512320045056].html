Operation Basalt

A sunset RHIB run from Wreck Ship Bivouac will hit Ancient Harbor from the south, recover the Radio Operator's Skell Tech RFID access module using less-than-lethal control where possible, then patch into the Golem Island Satellite Dish mainframe and pull Project Titan intelligence.

Start Point
Wreck Ship Bivouac south of Golem Island
Insertion
RHIB movement 1.3 km northwest to Ancient Harbor
Access Item
RTO-carried Skell Tech credentials module
Follow-On
Destroy heliport shipment if mainframe intel confirms it

Situation

The team has reached Golem Island, the home of Project Titan, after slipping past Legion drone swarms under DSI-CAMO and establishing a bivouac at a shipwreck south of the island at 2015 hours. Command wants the force to determine what Sentinel has been doing with Titan infrastructure since the invasion.

The immediate objective is to acquire network credentials from Ancient Harbor and use them to access the Golem Island Satellite Dish mainframe, which controls island communications and should contain actionable Project Titan intelligence.

Execution

Launch at sunset by RHIB, approach Ancient Harbor from the south, and work north through the site to locate the Radio Operator. Avoid hard gunfire until the access module is in hand because a stray round could destroy the credential device.

  • AlphaFind the Radio Operator at Ancient Harbor and recover his network access module.
  • BravoPatch into the Satellite Dish mainframe and download all actionable intelligence.
  • BranchIf the server confirms a fresh heliport shipment, move northeast, destroy it, then rest at the nearest bivouac.

Credential Handling

Use less-than-lethal control until the RTO module is secured. The RFID device is not bulletproof and can be ruined by stray fire.

Recon Limits

A strict no-fly zone and Legion drone patrols prevent meaningful ISR. Use Project Titan tacmap archives, binoculars, and miniature drones as terrain and signal conditions allow.

Server Lead

Mainframe intel should reveal a weapons and MRE shipment delivered to Golem Island Heliport one hour ago. Destroy that shipment before it transfers east.

Objective Alpha

Ancient Harbor

Infiltrate from the south, locate the Radio Operator, and obtain the Skell Tech access module needed for the dish breach.

Objective Bravo

Golem Island Satellite Dish

Access the top-side mainframe, download Project Titan intelligence, and identify any immediate follow-on targets.
